# cargo-crev

Source: https://github.com/crev-dev/cargo-crev (README)
Retrieved: 2026-03-10

---

`cargo-crev` implements Crev — a language and ecosystem agnostic, distributed code review system — integrated with Rust's cargo package manager.

## Key Capabilities

- **Security alerts**: Warn about untrustworthy crates and security vulnerabilities
- **Dependency analysis**: Display metrics about packages and identify bloat
- **Community-driven reviews**: Review suspicious dependencies and share findings
- **Trust networks**: Build webs of trust with other reputable users

## Installation & Support

Static binaries available through the project's releases page. Documentation on docs.rs. Active development; functional for current use.

## Community Engagement

The project recommends including a note in README files encouraging others to use cargo-crev to verify the trustworthiness of dependencies.

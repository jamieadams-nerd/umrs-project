# Red Hat Documentation Guidelines — Overview

Source: https://redhat-documentation.github.io/
Retrieved: 2026-03-12

## Guidelines for Red Hat Documentation

This page serves as a central repository for Red Hat's open-source documentation standards and conventions, made publicly available to benefit other open-source projects.

## Available Guides

**AsciiDoc Mark-up Conventions**
Covers proper styling techniques using AsciiDoc mark-up for Red Hat documentation content.

**Community Collaboration Guide**
Assists enterprise initiatives in coordinating with upstream communities on documentation matters.

**Red Hat Supplementary Style Guide for Product Documentation**
Complements the IBM Style Guide with specific style recommendations for Red Hat products. Includes a glossary section defining technical terminology and conventions used across Red Hat's documentation ecosystem.

**Modular Documentation Reference Guide**
Provides frameworks and examples for creating technical documentation using modular writing principles.

**Red Hat Peer Review Guide for Technical Documentation**
Establishes procedures and best practices for reviewing Red Hat's technical documentation.

**Vale at Red Hat**
Describes implementation of the Vale linting tool within Red Hat's documentation workflow.

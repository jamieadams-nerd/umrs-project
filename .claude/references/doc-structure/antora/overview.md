# Antora — Overview

Source: https://docs.antora.org/antora/latest/
Retrieved: 2026-03-12

## The Static Site Generator for Tech Writers

Antora is a documentation platform designed for technical writers to create and publish documentation to the web. Writers focus on authoring content in AsciiDoc, organize files into a standard project structure, and store them in git repositories. Antora then transforms this content into a website.

## Manage Docs as Code

Antora enables a "docs as code" approach, incorporating software development practices into documentation workflows:

- Version control system storage for content
- Separation of content, configuration, and presentation layers
- Automation for compilation, validation, verification, and publishing
- Reuse of shared materials (DRY principle)

## Getting Started Paths

**For newcomers:** Begin with "How Antora Can Help" to understand basic concepts, then try the Antora quickstart guide.

**For new projects:** Start by organizing your documentation files into Antora's expected structure, then create a playbook to configure documentation location and UI.

**For existing components:** Verify system requirements, install Antora, and generate your site.

**For content creation:** Learn AsciiDoc syntax to create documentation pages.

**For deeper understanding:** Explore how Antora works to understand the site generation pipeline.

# A Mathematical Theory of Communication

**Authors:** Claude E. Shannon
**Year:** 1948
**Published in:** The Bell System Technical Journal, Vol. 27, pp. 379–423, 623–656
**Source URL:** https://people.math.harvard.edu/~ctm/home/text/others/shannon/entropy/entropy.pdf

---

## Overview

Shannon's 1948 paper is the founding document of information theory. It established
a rigorous mathematical framework for quantifying information, defining the limits of
lossless and lossy data compression, and determining the theoretical maximum rate at
which information can be transmitted reliably over a noisy channel. Every modern
communication system, compression algorithm, and coding scheme traces its lineage
to this paper.

---

## Key Concepts and Theorems

### Entropy

Shannon defines the entropy H of a discrete random variable X with probability
distribution p(x) as:

    H(X) = -sum_x p(x) log2 p(x)

Entropy measures the average number of bits required to encode a symbol drawn from
the source. It is also the irreducible uncertainty in X — no lossless code can do
better than H bits per symbol on average.

Key properties:
- H >= 0 always
- H is maximized when all outcomes are equally probable (uniform distribution)
- H = 0 when one outcome has probability 1 (no uncertainty)
- H is additive for independent sources: H(X,Y) = H(X) + H(Y)

### Source Coding Theorem (Noiseless Channel Theorem)

For a stationary ergodic source with entropy H, there exists a prefix-free code
that compresses the source to arbitrarily close to H bits per symbol, and no lossless
code can compress below H. This establishes the fundamental limit of lossless
compression.

Consequences:
- Huffman coding achieves within 1 bit per symbol of H
- Arithmetic coding approaches H asymptotically
- Entropy is the right measure of information content

### Channel Capacity

For a noisy channel characterized by conditional probabilities P(Y|X), the channel
capacity C is:

    C = max_{p(x)} I(X; Y)

where I(X; Y) is the mutual information between input X and output Y:

    I(X; Y) = H(X) - H(X|Y) = H(Y) - H(Y|X)

Mutual information measures how much knowing Y tells us about X — it is the
"information shared" between two random variables.

### Noisy Channel Coding Theorem

For any rate R < C, there exist codes that allow transmission with arbitrarily low
error probability. For R > C, reliable communication is impossible. This is arguably
the most profound result in information theory — it guarantees reliable communication
over noisy channels is possible up to a theoretical limit.

### Conditional Entropy and Chain Rule

    H(X|Y) = -sum_{x,y} p(x,y) log2 p(x|y)

    H(X,Y) = H(X) + H(Y|X) = H(Y) + H(X|Y)

The chain rule decomposes joint entropy into marginal plus conditional, enabling
modular analysis of information in multi-variable systems.

### Relative Entropy (KL Divergence)

Though not named KL divergence by Shannon, the paper implicitly introduces the
asymmetric "distance" between probability distributions:

    D(P||Q) = sum_x p(x) log2 (p(x)/q(x))

This measures the inefficiency of encoding a distribution P using a code optimized
for Q. It is always >= 0 (Gibbs inequality), equals 0 iff P = Q.

### Typical Sequences and AEP

The Asymptotic Equipartition Property (AEP): for large n, sequences generated by
an ergodic source concentrate near the "typical set" of size approximately 2^{nH}.
This provides the conceptual foundation for source coding — only typical sequences
need short codes.

## UMRS Relevance

### RAG Indexing and Retrieval

Shannon entropy provides the theoretical foundation for understanding why certain
document representations are more efficient than others:

- **Entropy of term distributions:** A document corpus has an entropy over terms.
  Retrieval systems implicitly exploit this — rare terms (low probability, high
  self-information = -log2 p) carry more discriminative information. TF-IDF weights
  are an approximation of information content.
- **Mutual information for relevance:** I(query; document) is the ideal relevance
  measure. Modern embedding models learn representations that implicitly maximize
  this mutual information across training data.
- **Compression as index structure:** ChromaDB's vector store and embedding quantization
  are applications of source coding — trading some fidelity for storage efficiency.

### Canonical Representations and Deduplication

The source coding theorem tells us the minimum number of bits needed to represent any
object uniquely. This motivates:

- **Canonical forms:** If two security contexts encode the same access control decision,
  entropy analysis reveals redundancy. Canonical representations achieve near-entropy
  encoding.
- **Deduplication in posture signals:** When multiple signals encode overlapping
  information, mutual information quantifies the overlap. Removing redundant signals
  without information loss is an entropy-optimization problem.

### MLS Lattice Information Flow

Shannon's framework applies to MLS (Multi-Level Security) policy analysis:
- Information flow from low to high classifications can be modeled as channel capacity
  between security levels
- The Biba and Bell-LaPadula models enforce a "no-write-down, no-read-up" policy
  that can be analyzed as a zero-capacity channel from high to low
- Covert channels exploit residual channel capacity that policy does not eliminate;
  Shannon capacity bounds how much information can leak

### Contradiction Detection in Posture Catalog

When two posture signals give conflicting readings, the information-theoretic
interpretation is that the observed joint distribution is far from what the
model expects. Entropy of the contradiction set measures the "surprise" of the
system's state — high-entropy contradictions indicate a more anomalous system.

## Key Terminology

- **Entropy H(X):** Average bits of uncertainty/information in a random variable
- **Channel capacity C:** Maximum mutual information over all input distributions
- **Mutual information I(X;Y):** Information shared between two variables
- **Conditional entropy H(X|Y):** Remaining uncertainty about X after observing Y
- **Typical set:** The set of "likely" sequences, of size ~2^{nH}
- **Source coding:** Compression of information toward the entropy limit
- **Channel coding:** Adding redundancy to achieve reliable communication
- **Self-information:** -log2 p(x) — the surprise of observing outcome x

## Historical Notes

Shannon published this as a Bell Labs technical report while working on cryptography
and communication theory. It unified separate threads of work on telegraphy, telephony,
and statistical mechanics (the connection to Boltzmann entropy is explicit — Shannon
consulted von Neumann on the name "entropy"). The paper is unusually complete for a
foundational work — it contains both the noiseless and noisy channel theorems,
Huffman-style analysis, and continuous channel analysis (Gaussian channel, bandwidth-
limited AWGN).

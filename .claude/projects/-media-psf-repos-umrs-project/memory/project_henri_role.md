Henri's role parallels Herb's — independent validation authority, not downstream review.

- Henri validates his own Protected A/B/C mappings against TBS policy instruments
- Same authority pattern as Herb validating NIST control citations against SP 800-53
- Consultant for French cyber and unclassified security programs (Canadian side)
- Two independent validation functions (Herb: security controls, Henri: Canadian regulatory weight), both mandatory

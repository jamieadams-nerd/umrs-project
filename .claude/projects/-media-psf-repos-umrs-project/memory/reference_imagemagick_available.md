name: ImageMagick available
description: ImageMagick (convert binary) is installed on the system for image processing
type: reference
---

ImageMagick is installed — the `convert` binary is available for image resizing and format conversion.
Previously the project used Python Pillow for image work; now ImageMagick is also an option.

**How to apply:** When processing team portrait images or other image tasks, `convert` can be used
directly via Bash. Previous guidance was Pillow-only; both are now available.

Jamie's principle: "Extra work for us — easier for the operator."

Each country's catalog descriptions, handling guidance, and explanatory text must
read like it was written BY that country's government, FOR their operators. Not
translated American English. The style guides and terminology databases ensure we
match their voice.

Same discipline as Termium Plus for fr_CA — the corpus is consulted before any
terminology or phrasing decision.

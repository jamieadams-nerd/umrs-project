== Planned Purpose

`umrs-state` is the system state introspection subsystem for the UMRS project.
It captures point-in-time security posture snapshots — including kernel attributes,
SELinux enforcement status, FIPS mode state, and platform integrity indicators — and
serializes them for audit, monitoring, and reporting purposes. The crate follows a
strict layered separation model: a write path that collects and stores state, and a
separate read path consumed by CLI, TUI, and GUI display layers.

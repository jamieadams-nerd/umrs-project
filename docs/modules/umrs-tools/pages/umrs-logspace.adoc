== Planned Purpose

`umrs-logspace` is the audit trail and logging subsystem for the UMRS project.
It provides the foundation for structured, high-assurance audit event recording
aligned with NIST 800-53 AU controls and NSA RTB requirements. The crate is
intended to support tamper-evident log production, structured log formatting,
and integration with the system audit pipeline.

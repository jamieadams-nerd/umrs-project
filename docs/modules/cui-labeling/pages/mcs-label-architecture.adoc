= MCS Label Architecture
:description: How UMRS maps CUI categories to SELinux Multi-Category Security labels.

_This page is a placeholder for the MCS label architecture documentation. Content will be developed as the `umrs-label` tool is implemented._

== Overview

SELinux Multi-Category Security (MCS) provides the kernel mechanism for CUI labeling.
Each CUI category maps to one or more MCS category numbers (c0 through c1023).
The `mcstransd` daemon translates between human-readable category names and numeric labels using the `/etc/selinux/targeted/setrans.conf` configuration file.

== Overview

SELinux Multi-Category Security (MCS) provides the kernel mechanism for CUI labeling.
Each CUI category maps to one or more MCS category numbers (c0 through c1023).
The `mcstransd` daemon translates between human-readable category names and numeric labels using configuration files in `/etc/selinux/targeted/setrans.d/`.

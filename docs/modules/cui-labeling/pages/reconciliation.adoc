= Registry Reconciliation
:description: Comparison between the UMRS MCS mapping and the authoritative DoD CUI Registry and NARA canonical list.

This page documents the reconciliation between our existing CUI-LABELS.json mapping and the DoD CUI Registry at `dodcui.mil`.
The reconciliation was performed on 2026-03-21 by scraping all 113 DoD registry entries and comparing them against our 72-entry mapping.

IMPORTANT: The NARA registry is the authoritative source for UMRS banner markings, not the DoD registry.
Where DoD and NARA diverge, UMRS follows NARA.
Index groups (Defense, Intelligence, Immigration, etc.) are organizational navigation aids in both registries.
They never appear in banner markings.
Only category abbreviations appear in banners.

=== Analysis

The mismatches fall into three categories:

1. **Index group notation in DoD markings (most entries)**: The DoD registry uses index group abbreviations (PRIV, DEF, INTEL, IMMG, PAT, TRANS, FNC, LGL) as prefixes in its banner notation, producing strings like `CUI//INTEL/OPSEC`. This format is incompatible with NARA canonical markings and must not be adopted by UMRS.
+
Per the NARA registry, index groups are organizational navigation aids only. They never appear in banner markings. UMRS uses the NARA standalone form: `CUI//OPSEC`, not `CUI//INTEL/OPSEC`. This divergence from DoD notation is intentional and correct.

2. **Incorrect parent assignments in previous UMRS mapping (several entries)**: The previous UMRS mapping also used group-nested notation (`CUI//PRIVACY/CONTRACT`, `CUI//CTI/EXPT`, etc.). These were wrong for the same reason — index group names were embedded in markings. The UMRS Correct Marking column above corrects all of these to standalone form.

3. **Different index group assignment (4 entries requiring NARA verification)**: CTI, DCRIT, LMI, and BUDG have assignments that differ between the previous UMRS mapping and DoD. These require NARA cross-reference to confirm the standalone abbreviation is correct and no renaming has occurred.

CAUTION: Do not update the MCS mapping unilaterally based on DoD alone.
Cross-reference each HIGH priority mismatch against the NARA canonical registry at `https://www.archives.gov/cui/registry/category-list`.
Where DoD and NARA disagree on anything beyond index group nesting, flag for Jamie to make the call.

NOTE: The DoD registry's grouped notation (`CUI//INTEL/OPSEC`) is a DoD-internal convention for navigating their registry.
It is not valid NARA banner syntax.
UMRS systems that exchange data with DoD systems must use NARA standalone form in all banners and MCS labels.

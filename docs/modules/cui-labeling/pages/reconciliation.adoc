=== Analysis

The mismatches fall into three categories:

1. **Group abbreviation difference (7 entries)**: We used `PRIVACY` as the group abbreviation; DoD uses `PRIV`. This is a naming convention issue, not a structural disagreement. Resolution: adopt DoD's `PRIV` abbreviation.

2. **Standalone vs grouped (7 entries)**: We treated certain categories as top-level; DoD nests them under parent groups (Defense, Intelligence, Immigration, Patent, Transportation). The DoD grouping is authoritative for DoD systems. Resolution: adopt DoD's parent group assignments.

3. **Different parent group (4 entries)**: CTI/EXPT under CTI vs Export Control; DCRIT under CRIT vs Defense; LMI under GOVT vs Legal; BUDG under GOVT vs Financial. These require NARA cross-reference before resolution.

CAUTION: Do not update the MCS mapping unilaterally based on DoD alone.
Cross-reference each HIGH priority mismatch against the NARA canonical registry at `https://www.archives.gov/cui/registry/category-list`.
If DoD and NARA disagree, flag for Jamie to make the call.

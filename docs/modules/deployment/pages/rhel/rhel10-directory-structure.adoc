= UMRS Directory Structure
:description: Directory layout for UMRS configuration, binaries, and state on RHEL 10.

The following directory structure is planned for UMRS on RHEL 10. Paths follow
Linux FHS conventions and will have SELinux file context assignments once the
policy modules are complete.

== Planned Paths

[cols="1,2",options="header"]
|===
| Path | Purpose

| `/etc/umrs`       | Configuration files (JSON, policy references)
| `/usr/lib/umrs`   | Runtime support files
| `/usr/share/umrs` | Static data, label definitions, documentation
|===

== Files to Install

The following files will be placed into the UMRS directory structure:

* `umrs-mls-labels.json` → `/etc/umrs/`
* `umrs-mls-state.json` → `/etc/umrs/` (or `/var/lib/umrs/` — TBD)
* UMRS binaries → `/usr/bin/` or `/usr/lib/umrs/`
* SELinux policy modules → loaded via `semodule` (no permanent install path needed)

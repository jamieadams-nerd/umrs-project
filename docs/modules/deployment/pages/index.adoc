= Deployment
:toc: left
:description: Installing and configuring UMRS on RHEL 10 -- from hardware decisions through SELinux policy and UMRS installation.

This module covers everything needed to stand up a hardened UMRS platform.
Follow the five steps in order. Each step builds on the previous one.

[cols="1,3",options="header"]
|===
| Step | Description

| xref:rhel10/01-preinstall.adoc[Step 1 — Pre-Installation Planning]
| Partition layout, FIPS decision, network interface planning. Make these
  decisions before starting the RHEL 10 installer.

| xref:rhel10/02-install-choices.adoc[Step 2 — RHEL 10 Installation]
| Run the installer with FIPS mode enabled, custom partitioning, network
  configuration, admin user creation, and SELinux=enforcing.

| xref:rhel10/03-postinstall/index.adoc[Step 3 — Post-Installation Hardening]
| Five sub-tasks: STIG hardening (OpenSCAP), IMA/EVM integrity, kernel lockdown,
  dual-NIC setup, and temporary filesystem security.

| xref:rhel10/04-rust-packages.adoc[Step 4 — Package Installation]
| Required RPM packages for all UMRS systems plus the Rust development toolchain
  for build systems.

| xref:umrs/index.adoc[Step 5 — UMRS Installation]
| DAC users and groups, SELinux types and login mappings, FHS directory
  structure, install pipeline, configurations, and verification.
|===

= Security Concepts
:description: Foundational security concepts underlying the UMRS design.

This module covers the security principles and models that inform UMRS design decisions.
Understanding these concepts helps engineers and auditors evaluate why UMRS is built the
way it is.

Start here if you are new to mandatory access control or SELinux.

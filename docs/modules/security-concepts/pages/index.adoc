= Security Concepts
:description: Foundational security concepts underlying the UMRS design.

This module covers the security principles and models that inform UMRS design decisions.
Understanding these concepts helps engineers and auditors evaluate why UMRS is built the
way it is.

== Topics

* xref:security-model.adoc[Security Model Overview] — mandatory access control, CUI handling,
  and the three pillars of UMRS enforcement.

* xref:reference-monitor.adoc[Reference monitor design] — the tamperproof, non-bypassable
  access control kernel and its role in UMRS.

* xref:rtb-vnssa.adoc[NSA RTB and RAIN] — the NSA Reference Technical Brief requirements
  and how RAIN (Reference Architecture Implementation Notes) maps to UMRS.

* xref:integrity-and-provenance.adoc[Integrity and provenance] — what integrity assurance
  means in this context and how provenance is tracked.

* xref:truth-concepts.adoc[Truth and trust concepts] — ground truth, trust anchors, and
  the distinction between custody trust and content trust.

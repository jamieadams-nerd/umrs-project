== Overview

This module covers day-to-day administration of a UMRS-enabled system:
managing SELinux users and roles, applying policy updates, and maintaining
system configuration in an enforcing environment.

For deployment (initial setup and installation), see the
xref:deployment:index.adoc[Deployment module].

For operational monitoring and troubleshooting, see the
xref:operations:index.adoc[Operations module].

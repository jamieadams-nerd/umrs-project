= CPU Cryptographic Extensions
:description: Hardware cryptographic extensions — AES-NI, SHA extensions, AVX-512 VAES — and how to audit an ELF binary for their use.
:toc: left
:toclevels: 2

Modern processors include hardware extensions that accelerate cryptographic operations while maintaining constant-time execution properties. This page lists the relevant CPU extensions, describes their security implications, and explains how to audit a compiled ELF binary to determine which extensions it relies on.

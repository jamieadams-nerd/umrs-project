= File Operations — Secure API Inventory

For developers new to UMRS. You need to do something basic with files — here's the secure alternative from our stack. If the UMRS column says `(none)`, flag it on the task board rather than rolling your own.

Every read in UMRS is capped at a documented byte limit — see xref:bounded-reads.adoc[Bounded reads] for the canonical list.

== Opening and reading files

[cols="2,3,3,1,1",options="header"]
|===
| Task | Traditional | UMRS secure method | Evidence | Trust

| Open a file for reading (TOCTOU-safe, fd-anchored)
| `File::open(path)` — re-resolves path on every follow-up call
| `umrs_selinux::secure_dirent::SecureDirent::from_path` — opens once; all further reads use the fd
| no
| —

| Read a file as bytes
| `std::fs::read`
| (none)
| —
| —

| Read a file as text
| `std::fs::read_to_string`
| (none)
| —
| —

| Read a file from `/proc`
| `std::fs::read_to_string("/proc/...")`
| `umrs_platform::kattrs::SecureReader<ProcfsText>::read_generic_text`
| no
| T1

| Read a file from `/sys`
| `std::fs::read_to_string("/sys/...")`
| `umrs_platform::kattrs::SecureReader<SysfsText>::read_generic_text`
| no
| T1
|===

== SELinux file labels (`security.selinux` xattr)

UMRS does not link against `libselinux`. Label reads are pure Rust via `fgetxattr(2)`. The UMRS function names mirror the libselinux trio so anyone reaching for `getfilecon(3)` lands on the right call.

[cols="2,3,3,1,1",options="header"]
|===
| Task | Traditional | UMRS secure method | Evidence | Trust

| Read a file's SELinux label (follows symlinks)
| `getfilecon(3)` or `getxattr(2, "security.selinux")`
| `umrs_selinux::utils::get_file_context`
| no
| —

| Read the label without following symlinks
| `lgetfilecon(3)` or `lgetxattr(2, "security.selinux")`
| `umrs_selinux::utils::lget_file_context`
| no
| —

| Read the label from an open fd
| `fgetfilecon(3)` or `fgetxattr(2, "security.selinux")`
| `umrs_selinux::utils::fget_file_context`
| no
| —

| Set a file's SELinux label
| `setfilecon(3)` / `setxattr(2, "security.selinux")`
| (none)
| —
| —
|===
